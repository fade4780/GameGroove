# GameGroove - Crowdfunding 2.0 for Indie Games

GameGroove is a modern crowdfunding platform specifically designed for indie game developers and gaming enthusiasts. It combines the best aspects of crowdfunding with community engagement and playable demos.

## Features

- **Playable Demos**: Every project must include a playable demo (WebGL/EXE)
- **Community Engagement**: Players can influence game development through voting and feedback
- **Gamified Backing**: Backers receive in-game rewards and special recognition
- **Transparent Funding**: Detailed breakdown of how funds are being used
- **Verified Studios**: Special section for vetted game development studios

## Tech Stack

- React 18
- TypeScript
- Styled Components
- Framer Motion
- React Router
- Axios

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/gamegroove.git
cd gamegroove
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```
